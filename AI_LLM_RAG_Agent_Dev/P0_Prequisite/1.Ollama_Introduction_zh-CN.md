## 什么是 Ollama？

[Ollama](https://ollama.com) 是一款开源工具，让你在本地机器上轻松运行各种大语言模型（LLM）。无需复杂的配置，一条命令即可下载、运行和管理开源模型，所有数据都保留在本地，隐私无忧。

### 为什么选择 Ollama？

- **零配置上手** - 无需折腾 CUDA、Python 环境，开箱即用
- **隐私优先** - 数据完全本地化，不上传云端
- **硬件友好** - 自动检测 GPU，无显卡也能用 CPU 运行
- **生态丰富** - 支持 100+ 开源模型，从轻量级到超大规模

## 安装

### macOS

```bash
brew install ollama
```

或从 [官网](https://ollama.com/download) 下载安装包。

### Linux

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Windows

直接从 [官网下载](https://ollama.com/download) 安装程序。

### Docker

```bash
docker pull ollama/ollama
```

## 快速开始

### 1. 启动服务

```bash
ollama serve
```

服务默认运行在 `http://localhost:11434`。

### 2. 运行你的第一个模型

```bash
ollama run llama3.2
```

首次运行会自动下载模型（Llama 3.2 约 2GB）。下载完成后直接进入交互式对话：

```
>>> 你好
你好！有什么我可以帮你的吗？

>>> /bye
```

用 `/bye` 退出对话。

### 3. 常用命令速查

| 命令 | 作用 |
|------|------|
| `ollama list` | 查看已下载的模型 |
| `ollama pull <model>` | 下载模型（不运行）|
| `ollama run <model>` | 下载并运行模型 |
| `ollama rm <model>` | 删除模型 |
| `ollama ps` | 查看正在运行的模型 |
| `ollama stop <model>` | 停止指定模型 |

## 推荐模型

Ollama 支持数百个模型，访问 [ollama.com/library](https://ollama.com/library) 查看完整列表。

### 快速选择指南

| 用途 | 推荐模型 | 大小 | 命令 |
|------|---------|------|------|
| **日常对话** | Llama 3.2 | 2.0GB | `ollama run llama3.2` |
| **中文场景** | Qwen 2.5 | 4.7GB | `ollama run qwen2.5` |
| **代码助手** | CodeLlama | 3.8GB | `ollama run codellama` |
| **深度推理** | DeepSeek-R1 | 4.7GB | `ollama run deepseek-r1` |
| **图像理解** | Llama 3.2 Vision | 7.9GB | `ollama run llama3.2-vision` |
| **文本嵌入** | nomic-embed-text | 274MB | `ollama pull nomic-embed-text` |

### 模型版本说明

模型通常有多个版本（称为 "tag"），用冒号分隔：

```bash
ollama run llama3.2        # 默认版本（推荐）
ollama run llama3.2:1b     # 轻量版，速度更快
ollama run llama3.2:3b     # 标准版
```

> **内存要求参考**：7B 模型约需 8GB 内存，13B 约需 16GB，33B 约需 32GB。

## API 调用

Ollama 启动后会提供 HTTP API，方便集成到其他应用：

### 聊天接口

```bash
curl http://localhost:11434/api/chat -d '{
  "model": "llama3.2",
  "messages": [{"role": "user", "content": "你好"}]
}'
```

### 生成接口（流式响应）

```bash
curl http://localhost:11434/api/generate -d '{
  "model": "llama3.2",
  "prompt": "写一首关于春天的诗"
}'
```

### Python 示例

```python
import requests

response = requests.post('http://localhost:11434/api/chat', json={
    'model': 'llama3.2',
    'messages': [{'role': 'user', 'content': '你好'}],
    'stream': False
})
print(response.json()['message']['content'])
```

更多 API 详情参考 [官方文档](https://github.com/ollama/ollama/blob/main/docs/api.md)。

## 进阶功能

### 自定义模型（Modelfile）

通过 Modelfile 创建定制模型：

```dockerfile
FROM llama3.2

SYSTEM """你是一个专业的编程助手，擅长 Python 和 JavaScript。"""

PARAMETER temperature 0.7
```

创建并运行：

```bash
ollama create my-assistant -f Modelfile
ollama run my-assistant
```

### 多行输入

在交互模式下用 `"""` 包裹多行内容：

```
>>> """
... 第一行
... 第二行
... 第三行
... """
```

## 常见问题

**Q: 如何彻底退出 Ollama？**

交互模式输入 `/bye`，或终端按 `Ctrl+C`。要停止后台服务：

```bash
# macOS/Linux
pkill ollama

# Windows PowerShell
Get-Process ollama | Stop-Process
```

**Q: 模型下载到哪里了？**

- macOS: `~/.ollama/models/`
- Linux: `/usr/share/ollama/.ollama/models/`
- Windows: `C:\Users\<用户名>\.ollama\models\`

**Q: 如何指定使用 CPU/GPU？**

Ollama 默认优先使用 GPU。如需强制 CPU：

```bash
OLLAMA_GPU_OVERHEAD=1 ollama run llama3.2
```
